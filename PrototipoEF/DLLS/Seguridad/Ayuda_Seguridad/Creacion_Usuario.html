Mantenimiento Usuario

El formulario consta de los botones Nuevo,Guardar,Modificar,Eliminar y Salir

Para crear un nuevo Usuario,se debe dar click en el boton Nuevo, y luego llenar todos los campos,al realizar esto se da click en el boton Guardar,y el usuario se a creado.


Para modificar un Usuario,se debe dar ingresar el id del usuario y luego dar click en el boton Buscar,al realizarlo si el usuario existe se llenaran todos lo campos con los datos del usuario para luego poder modificarlos,al realizar la modificacion se da click en el boton Guardar y los cambios se han realizado.


Para eliminar un Usuario se debe ingresar el Id del usuario y luego dar click en el boton Buscar,al realizarlo si el usuario existe se llenaran todos los campos con los datos del usuario, luego se dara click en el boton Eliminar y el usuario se a eliminado.
